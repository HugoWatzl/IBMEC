# 📚 Anotações, Códigos e Projetos Avaliativos(AP1 e AP2) das Materias da Faculdade

Este repositório contém **anotações de aulas** e **códigos** desenvolvidos por mim durante os cursos da faculdade. O objetivo principal é centralizar todo o material de estudo para facilitar revisões, aprimorar o aprendizado.

---

## 🚀 Conteúdo

Este repositório está dividido em várias pastas e arquivos organizados por disciplina ou tema, contendo:

- **Anotações**: Resumos e detalhes de tópicos abordados nas aulas.
- **Códigos**: Implementações de algoritmos, exercícios e projetos de programação desenvolvidos durante o curso.
- **Projetos**: Materiais que envolvem projetos de maior escala, com documentação detalhada.

---

- **Ferramentas de Desenvolvimento**: VSCode, Git, GitHub, intelj
