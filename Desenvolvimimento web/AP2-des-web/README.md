# 🏃‍♂️ AP2 - Desenvolvimento Web

Sistema web desenvolvido para busca de atletas utilizando uma API externa, com autenticação simples baseada em verificação de senha por caracteres.

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como parte da disciplina de Desenvolvimento Web.  
A aplicação permite:

- 🔍 Buscar atletas por nome através de uma API
- 🔐 Realizar autenticação com verificação de senha caractere por caractere
- 🌐 Exibir resultados dinamicamente na interface web

---

## 🚀 Funcionalidades

- Busca de atletas em tempo real
- Consumo de API REST
- Validação de login com checagem de senha
- Interface simples e responsiva

---

## 🛠️ Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- API REST
